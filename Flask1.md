1. What is Flask?

Flask is a lightweight Python web framework used to build web applications and APIs. It follows the WSGI standard and is known for its simplicity and flexibility.

2. Who created Flask?

Flask was created by Armin Ronacher as part of the Pallets Projects ecosystem.

3. Is Flask a microframework?

Yes. Flask is a microframework because it provides only essential features like routing and request handling without enforcing heavy dependencies.

4. What language is Flask written in?

Flask is written in Python.

5. What is WSGI?

WSGI stands for Web Server Gateway Interface, a standard interface between Python web applications and web servers.

6. What are the main components of Flask?

Main components include:

Routing

Templates

Request handling

Response handling

Sessions

Extensions

7. What template engine does Flask use?

Flask uses Jinja2 as its default template engine.

8. What command runs a Flask application?
flask run
9. What is app.route()?

@app.route() is a decorator used to define URL routes for functions.

Example:

@app.route("/")
def home():
    return "Hello"
10. What is the default port of Flask?

Default port is 5000.
